# StyleSelf AI 👗✨

An AI-powered fashion styling web app that helps users:

- Generate outfits based on occasion and budget  
- Complete a look from a single clothing item  
- Get real product images and shopping links  
- Save favorite outfits  
- Personalize styling using user profile  

## Tech Stack

- Frontend: React (Vite)  
- Backend: Node.js + Express  
- AI: Groq API (LLaMA)  
- Database: MongoDB  

## Features

- Buy Outfit  
- Complete Look  
- Wardrobe Styling  
- Profile Personalization  
- Saved Outfits
